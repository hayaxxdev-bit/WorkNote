# 📝 MyNotepad Online

Notepad online untuk mencatat segala hal - project, daily notes, pekerjaan, ide, dan lainnya!

## ✨ Fitur

- 📝 **Multiple Notes** - Buat banyak catatan sekaligus
- 🏷️ **Kategori** - Atur catatan berdasarkan kategori (Project, Daily, Work, Personal, Ideas)
- 🔍 **Pencarian** - Cari catatan dengan mudah
- 🎨 **Formatting** - Bold, italic, underline, strikethrough, list, checkbox
- 📅 **Insert Otomatis** - Masukkan tanggal dan waktu dengan satu klik
- 🌙 **Dark Mode** - Tema gelap untuk kenyamanan mata
- 💾 **Auto Save** - Catatan tersimpan otomatis di browser
- 📤 **Export** - Download catatan sebagai file markdown
- 📱 **Responsive** - Bisa diakses dari desktop dan mobile

## 🚀 Deployment

### Deploy ke Vercel

1. Fork repository ini ke GitHub Anda
2. Buka [Vercel](https://vercel.com)
3. Import project dari GitHub
4. Deploy!

Atau deploy langsung:

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/YOUR_USERNAME/my-notepad-online)

### Run Locally

```bash
# Clone repository
git clone https://github.com/YOUR_USERNAME/my-notepad-online.git

# Masuk ke direktori
cd my-notepad-online

# Install Vercel CLI (opsional)
npm i -g vercel

# Run development server
vercel dev
